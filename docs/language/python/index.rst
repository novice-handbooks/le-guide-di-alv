Python - Un linguaggio per mille usi
====================================

.. note::

  Queste pagine sono nate per ospitare materiale per promuovere l'apprendimento
  del linguaggio Python.
  Lo scopo finale è quello di riportare le informazioni in modo che il
  lettore possa crescere nell'apprendimento.

.. attention::

  Gli argomenti inseriti in fase di creazione però seguono un ordine
  che dipende dalle esigenze di apprendimento dell'autore.
  È quindi possibile che le informazioni non siano inizialmente fruibili
  da una platea senza conoscenza pregressa.
